# 🏆 Ideal World Cup (이상형 월드컵)

* 익명 참여형 이상형 월드컵 웹 서비스
* 본 프로젝트는 기획부터 DB/보안 설계, 프론트엔드/백엔드 구현까지 **풀스택으로 단독 수행**한 사이드 프로젝트입니다.

<br/><br/>

## 📌 프로젝트 기획 배경

그동안 내가 느낀 대중들은 MBTI 테스트처럼<br/>
가볍게 참여하고 친구에게 공유하는 콘텐츠에 높은 반응을 보이는 듯 했다.<br/>

나 역시 그런 흐름에 익숙했다.<br/>

그래서 참여와 공유가 자연스럽게 이어지는 구조의 서비스를 직접 만들어보고 싶다는 생각을 했고<br/>
그 고민의 결론은, 다양한 주제로 토너먼트 월드컵 게임을 즐기는 사이트였다.<br/>

사용자가 결과를 공유할 수 있도록 설계하여<br/>
하나의 참여가 또 다른 참여로 이어지도록 기획했다.<br/>

아직은 로직을 구현하는 단계이지만, 이러한 생각 끝에 이 서비스가 출발하게 되었다.

<br/><br/>
